Markdown Cheat Sheet
Basic Syntax
# Heading 1 Creates a heading level 1
## Heading 2 Creates a heading level 2
**bold text** Creates bold text
*italic text* Creates italic text
[link text](https://example.com) Creates a link
![alt text](image.jpg) Inserts an image
- List item Creates a bullet list item
1. Ordered item Creates a numbered list item
Code Blocks

For inline code, use backticks: `code`

For code blocks, use triple backticks with optional language name:

```python
def hello_world():
    print("Hello, world!")
```
